gwreports - geekwright Reports

Purpose
=======
A MySQL reporting module for XOOPS and ImpressCMS


Installation
============
Copy gwreports directory to the modules directory of your 
site, then install like any other module in your site's 
administration area.


Usage
=====
gwreports is a flexible and powerful tool with many options. Complete 
documentation, including tutorials, is available at geekwright.com


Notes
=====
This module has been tested in ImpressCMS versions 1.2.7 and 1.3.4,
and in Xoops version 2.5.5.
